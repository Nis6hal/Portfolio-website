# Nischal Bhandari — Portfolio

Personal portfolio website with a Node.js/MongoDB backend and a protected admin panel.

**Live site:** [nischal-bhandari.com.np](https://nischal-bhandari.com.np)

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **Hosting:** GitHub Pages (frontend) + Render (backend)
- **Email:** Resend

---

## Features

- Dynamic content loaded from API
- Protected admin panel for managing projects, skills, and content
- Contact form with email verification
- Responsive design with mobile support
- Interactive chatbot

---

## Admin Panel

Accessible at `/admin.html` — requires login credentials.

Manage projects, skills, hero text, about section, contact info, and section visibility toggles.

---

## Environment Variables

All secrets are stored as environment variables and never committed to the repository.
